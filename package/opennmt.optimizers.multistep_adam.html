opennmt.optimizers.multistep_adam module¶

Optimizer variants which make it possible to use very large batch sizes with limited GPU memory. Optimizers in this module accumulate the gradients for n batches, and call the optimizer’s update rule every n batches with the accumulated gradients. See [Saunders et al., 2018](https://arxiv.org/abs/1805.00456) for details.
